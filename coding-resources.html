Coding resources

Courses & General Learning | Documentation | YouTube | Cheatsheets | UI/UX | Miscellaneous

Here you can find a repository of online resources that I consider good or useful when it comes to learning how to code or just useful resources in general. These are for anyone who wishes to learn to code and wants to know where to get started as well as for my own personal use. I recommend bookmarking the ones you find most useful.

A Note For Fellow Learners

My number one piece of advice would be to always prefer building projects over watching tutorials. If you put yourself in a position where you have to solve a problem you don't know how to solve, you will be forced to learn how to solve it. Use learning resources and tutorials as a way to gain knowledge but remember that the truest implications of something you learn will only be revealed in a problem setting. And regardless of what level you're at, there is always something you can build, whether it's building a simple program that adds 2 numbers or a piece of large-scale software. In either, there is something you can learn and you can google your way to success. 10 projects under your belt will always be better than 100 tutorials watched because of this.

Happy coding!

Courses & General Learning

A good place to start if new.

  • Codecademy - Various free and diverse courses for different programming topics and roles. Good for starting out.
  • The Odin Project - Probably the best course out there for learning web development.
  • roadmap.sh - A site that has several different learning roadmaps for different roles and languages. Has other external resources and good to follow along with when learning.
  • W3Schools - Lots of documentation for different programming languages with short tutorials and examples.
  • Codewars - A website where you can undertake coding problems and challenges to test your skills. Good form of hands-on learning.
  • ChatGPT - Good source of quick, tailored, on-the-fly responses to things you want to know more about. Good for programming.

Documentation

Comprehensive documentation of different programming languages that I have used at some point. Good if you want nitty gritty, detailed knowledge. If something you are interested in is not in this list, you can easily google it (e.g "C# documentation").

  • MDN Web Docs - The best piece of documentation for development on the web. All you need to know about HTML, CSS, JavaScript, HTTP, Web API's, Web Accessibility and more.
  • Python - The official Python documentation.
  • Java - The official Java documentation.
  • Node.js - The official Node.js documenation.
  • PHP - The official PHP documentation.
  • Git Documentation - The official documentation for Git, the version control software.

YouTube

Good all-around Youtube channels for learning how to code.

  • freeCodeCamp - Comprehensive tutorials on different programming languages and projects.
  • BroCode - Comprehensive and no-nonsense coding tutorials.
  • Fireship - Short form tutorials as well as informational videos about the tech and coding space.
  • Web Dev Simplified - One of the best Youtube channels for learning web development. Simple, no-nonsense, easy to follow along and highly informational.
  • Mosh Hamedani - Comprehensive tutorials similar to freeCodeCamp.
  • TraversyMedia - Crash courses on several programming languages and subjects.
  • Kevin Powell - Pragmatic tutorials when it comes to learning and mastering CSS.

Cheatsheets

Good for if you forget something or just want the fastest possible answer to niche subjects or anything in general. Here are some that I have used or would use.

  • Git Cheatsheet - Cheatsheet for Git, the version control software.
  • CSS Cheatsheet - A cheatsheet for CSS with interactive examples.
  • HTML Cheatsheet - A good HTML cheatsheet to tell you about tags, attributes and syntax.
  • Web Security Cheatsheet - A web security cheatsheet I use to improve the security of my web projects.
  • MIME Types Cheatsheet - If you are ever developing for the web like me and need to know the MIME type for different files.
  • Big O Cheatsheet - If you are studying data structures & algorithms and need to know about space/time complexity.

UI/UX

Resources for when working with user interfaces and design. Sources for good images, inspiration, design tools and more.

  • Google Fonts - A font library of easily downloadable and free to use fonts. Also comes with an icon library.
  • Figma - Powerful, industry-standard and free to use design software.
  • SVGOMG - A tool for optimising SVG files.
  • Minimal Gallery - A collection of beautiful, slick and modern websites to use as inspiration.
  • ManyPixels - An illustration library for minimal but beautiful illustrations and SVGs.
  • Color Hunt - A source for color palettes.
  • Designspiration - A collection of images and graphic design pieces to use as inspiration.
  • GifCities - An archive of old GIFs from the 90s/00s web. Many that I have used on this website.
  • A.N Lucas' Buttons - A collection of 88x31 buttons, a hallmark feature of websites from the old web.
  • 98.css - A design system for creating Windows 98 inspired UIs.
